## Usage

Skills trigger automatically when you discuss Lindy topics. For example:

- "Help me set up Lindy" -> triggers `lindy-install-auth`
- "Debug this Lindy error" -> triggers `lindy-common-errors`
- "Create a Lindy agent" -> triggers `lindy-core-workflow-a`
