## Usage

Skills trigger automatically when you discuss Gamma topics. For example:

- "Help me set up Gamma" -> triggers `gamma-install-auth`
- "Debug this Gamma error" -> triggers `gamma-common-errors`
- "Deploy my Gamma integration" -> triggers `gamma-deploy-integration`
